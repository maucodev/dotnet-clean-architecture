# Clean Architecture

## Overview
This project is a comprehensive guide to the clean architecture, implemented in C#. It serves as an educational tool for those looking to learn more about clean architecture and their implementation.

## Based on a Course
This project is based on the [Pragmatic Clean Architecture course by Milan Jovanovic.](https://www.milanjovanovic.tech/pragmatic-clean-architecture) The course provides a deep understanding of clean architecture using C#. This project is an implementation of the concepts taught in the course.

## Structure
The solution is divided into some main projects, each representing a fundamental layer:

1. **Bookify.Domain**: This project contains the implementation for the domain layer. It includes concepts like Entities, Value objects, Domain events, Domain services, and so one.
2. **Bookify.Application**: This project contains the implementation for the application layer. It includes concepts like CQRS pattern and MediatR, Cross-cutting concerns, Dependency inyection configuration, and so one.
